- `用于自动以数据为中心的开发的协作演进策略 <https://arxiv.org/abs/2407.18690>`_
  Co-STEER 是一种解决以数据为中心的开发 (AD2) 任务的方法，并突出了其主要挑战，这些挑战需要专家级的实现（即从实践中学习领域知识）和任务调度能力（例如，从更容易的任务开始以获得更好的整体效率），而这些领域以前的工作在很大程度上忽略了。我们的 Co-STEER 智能体通过我们的演进策略增强其领域知识，并通过收集和使用特定领域的实践经验来提高其调度和实现技能。通过更好的调度，实现变得更快。同时，随着实现反馈变得更加详细，调度准确性也得到提高。这两种能力通过实践反馈共同成长，实现了协作演进的过程。
