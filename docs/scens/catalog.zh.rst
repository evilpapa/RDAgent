场景列表
=========================

在数据驱动的两大核心领域——模型实现与数据构建——我们的系统服务于两类主要角色：🧬copilot 与 🤖agent。

- 🧬copilot 按照人类指令自动化重复性任务。
- 🤖agent 更具自主性，主动提出新想法以获得更优结果。
